KH Number of Normal vs Gameplay Cutscenes

This is a graph that represents the number of Gameplay and Normal cutscenes within the first two main Kingdom Hearts titles. We wanted to create
this graph to see just what the proportion of Gameplay and Normal cutscenes were in the games after we had fixed our code for the game scripts. What
we found is that in Kingdom Hearts 1 the amount of Gameplay and Normal cutscenes are very similar, being only 6 cutscenes apart. But when it comes
to Kingdom Hearts 2, this is changed dramatically as there are over 300 Normal cutscenes and only 91 Gameplay cutscenes.
